# 博客图表生成说明

## 需要的图表类型

### 1. 系统架构图
**用途**: 展示论文查重系统的整体架构
**工具推荐**: 
- Draw.io (https://app.diagrams.net/)
- Lucidchart
- Visio

**内容**: 
- Web前端、命令行工具、API接口
- 业务服务层
- 算法层（余弦相似度、编辑距离、Jaccard相似度）
- 工具层（性能监控、文件处理、异常处理）

### 2. 算法流程图
**用途**: 展示文本相似度计算流程
**工具推荐**: 
- Draw.io
- ProcessOn
- Visio

**内容**:
- 主程序流程图
- 文本相似度计算流程图
- 文本预处理流程图

### 3. 性能分析图
**用途**: 展示性能瓶颈分析结果
**工具推荐**: 
- Excel
- Google Sheets
- 在线图表工具

**内容**:
- 性能瓶颈分布饼图
- 各算法性能对比柱状图
- 优化前后性能对比图

### 4. 测试覆盖率图
**用途**: 展示测试覆盖率分析结果
**工具推荐**: 
- Excel
- JaCoCo报告截图
- 在线图表工具

**内容**:
- 各模块测试覆盖率柱状图
- 测试用例统计饼图
- 测试通过率趋势图

### 5. Web界面截图
**用途**: 展示前端界面设计
**工具推荐**: 
- 浏览器截图
- 设计工具截图

**内容**:
- 主界面截图
- 文件上传界面
- 结果展示界面

## 图表生成步骤

### 步骤1: 准备数据
根据博客文章中的数据准备图表数据：

**性能分析数据**:
- 文本预处理瓶颈: 40%
- 算法计算瓶颈: 35%
- 内存使用瓶颈: 25%

**算法性能数据**:
- 余弦相似度: 100ms
- Jaccard相似度: 120ms
- 编辑距离: 2000ms

**测试覆盖率数据**:
- CosineSimilarity: 91%
- LevenshteinSimilarity: 91%
- JaccardSimilarity: 90%
- PlagiarismDetectionService: 85%
- PlagiarismController: 79%
- 总计: 88%

### 步骤2: 选择工具
推荐使用以下工具：

1. **Draw.io** (免费，功能强大)
   - 网址: https://app.diagrams.net/
   - 适合: 架构图、流程图

2. **Excel/Google Sheets** (免费，易用)
   - 适合: 柱状图、饼图、性能分析图

3. **在线图表工具**
   - Chart.js
   - D3.js
   - 适合: 交互式图表

### 步骤3: 生成图表
按照以下顺序生成图表：

1. **系统架构图** → 使用Draw.io
2. **算法流程图** → 使用Draw.io
3. **性能分析图** → 使用Excel
4. **测试覆盖率图** → 使用Excel
5. **Web界面截图** → 使用浏览器截图

### 步骤4: 插入博客
将生成的图表保存为图片格式（PNG/JPG），然后插入到博客文章中。

## 图表样式建议

### 颜色方案
- 主色调: 蓝色系 (#2196F3)
- 辅助色: 绿色系 (#4CAF50)
- 警告色: 橙色系 (#FF9800)
- 错误色: 红色系 (#F44336)

### 字体建议
- 标题: 16px, 粗体
- 标签: 12px, 常规
- 数据: 14px, 粗体

### 布局建议
- 图表宽度: 800px
- 图表高度: 600px
- 边距: 20px

## 注意事项

1. **图表质量**: 确保图表清晰、易读
2. **数据准确性**: 确保图表数据与博客内容一致
3. **格式统一**: 保持所有图表的风格一致
4. **文件大小**: 控制图片文件大小，便于网页加载
5. **版权问题**: 使用免费工具或自己设计的图表
